# Customer Churn Prediction

## Objective
Predict which customers are likely to leave a company using machine learning.

## Tools
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Machine Learning Models
- Logistic Regression
- Random Forest
- XGBoost

## Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Business Impact
Reduce customer churn and increase revenue through predictive analytics.
