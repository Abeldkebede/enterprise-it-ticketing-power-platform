# System Architecture

## Components

- Power Apps Frontend
- SharePoint Online Data Storage
- Power Automate Workflows
- Power BI Reporting Dashboard

## Workflow

1. User submits ticket
2. Ticket stored in SharePoint
3. Power Automate triggers notifications
4. Technician updates ticket
5. Power BI updates reporting dashboard

## Security

- Microsoft 365 Authentication
- Role-based access control
- Audit logging
